# Pi05 (π₀.₅)

## 定义

Physical Intelligence 推出的 [[VLA]] 系列第二代——在 π₀ 的 [[Flow Matching]] action head 基础上引入更强的预训练 backbone 和跨实体微调，是当前开源/可获取 VLA baseline 的事实标准。

## 核心要点

1. **架构**: 3B [[VLM]] backbone + flow-matching action expert，输出 [[Action Chunking|chunked actions]]
2. **训练**: 大规模 embodied pretraining + 任务特化 SFT
3. **基线地位**: 几乎所有 2025–2026 VLA / WAM 论文都拿它作为对比下界
4. **弱点**: 长程记忆依赖任务（如 RMBench Press Button）表现接近 0，缺乏跨步语义状态维持能力

## 代表工作

- [[WLA]]: 在 RoboTwin / LIBERO / RMBench 上全面超越 π₀.₅
- [[GRAIL]]: 把 π₀.₅ 风格的 egocentric VLA 用合成视频训练
- [[AdaWAM]]: 作为 RoboTwin 2.0 / LIBERO / 真机的核心 action-only 基线对照
